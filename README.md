# poc_fpm


Trivial proof of concept - using [fpm](https://github.com/jordansissel/fpm) to build packages via docker

Examples include : 

|--------|--------|
| alpine | latest |
| ubuntu | bionic |
| ubuntu | cosmic |
| ubuntu | disco  |
| ubuntu | xenial |
| centos | 6      |
| centos | 7      |
|--------|--------|
